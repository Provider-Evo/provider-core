# qwen/mvp 模块

Qwen 平台的 MVP（最小可行产品）测试模块，用于验证 Qwen web 流程的基本功能。

## 目录结构

```
mvp/
├── chat.py          # 异步聊天烟雾测试
└── env.py           # 环境变量和凭证管理
```

## 核心功能

### chat.py

独立的异步聊天烟雾测试，验证当前 Qwen web 流程的基本功能。

**主要功能：**
- 登录认证
- 创建新对话
- 发送聊天消息
- 接收流式响应

### env.py

环境变量和凭证管理，提供测试所需的配置信息。

## 依赖关系

- 依赖 `src.platforms.qwen.core` 模块
- 使用 `aiohttp` 进行异步 HTTP 请求
- 依赖 `src.logger` 进行日志记录

## 注意事项

- 仅用于开发和测试环境
- 需要有效的 Qwen 账号凭证
- 不要在生产环境中使用
