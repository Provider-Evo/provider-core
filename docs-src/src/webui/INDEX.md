# src/webui 索引

- `app.py`：独立 WebUI 应用工厂。
- `config_schema.py`：WebUI 配置与导出模型。
- `dependencies.py`：请求依赖与默认设置解析。
- `logs_ws.py`：WebSocket 日志广播器。
- `routes.py`：WebUI 路由注册入口。
- `server.py`：独立 WebUI 服务器与线程化服务器。
- `middleware/`：WebUI 中间件。
- `routers/`：分路由处理器。
- `schemas/`：摘要序列化结构。
- `services/`：摘要服务。
- `utils/`：导出等工具函数。
