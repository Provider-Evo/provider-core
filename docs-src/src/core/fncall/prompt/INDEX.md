# prompt 模块

函数调用提示词构建包，提供提示词注入和模板管理功能。

## 目录结构

```
prompt/
├── __init__.py      # 模块初始化，重导出 echotools.fncall.prompt
├── inject.py        # 提示词注入
├── history.py       # 历史记录管理
└── templates.py     # 模板管理
```

## 核心功能

### 提示词注入

- `inject_fncall()`: 注入函数调用到提示词

### 历史记录管理

- 提供对话历史记录管理功能

### 模板管理

- 提供提示词模板管理功能

## 依赖关系

- 重导出 `echotools.fncall.prompt` 模块
- 被路由模块和平台适配器使用

## 注意事项

- 提示词注入支持多种格式
- 模板管理支持动态加载
- 历史记录管理支持上下文维护
