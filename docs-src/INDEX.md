# docs-src 索引

## 目录说明

- `DOCS-SRC-GUIDE.md`：docs-src 镜像文档树编写规范。
- `TESTS-GUIDE.md`：tests 镜像与测试编写规范。
- `MISTAKES.md`：常见错误和解决方案。
- `AGENTS.md`：历史根级 AGENTS 的镜像文档。
- `src/`：源码镜像文档。
- `tests/`：测试目录镜像文档。
- `.agents/`：skill 目录镜像文档。
- `.scripts/`：保留脚本目录与历史 AGENTS 镜像文档。
- `template/`：模板目录镜像文档。
