# rules 索引

本目录保留原 `.agents/rules/` 的完整规则内容，已作为 skill 的参考资料迁移进来。

## 核心规则

- `CODE_GUIDE.txt`：全局代码规范。
- `AGENTS_MD_GUIDE.md`：AGENTS 文档规范。
- `README_GUIDE.md`：README 编写规范。
- `REVERSE_GUIDE.md`：逆向相关说明。
- `SYSTEMINFO.md`：环境与系统信息。
- `CLAUDE.md`：综合工作规范。

## 专项规则

- `Frontend/CLAUDE.md`：前端开发规范。
- `Frontend/STYLE-DETAIL.txt`：前端样式细节。
- `General/CLAUDE.md`：通用规则。
- `Bug/CLAUDE.md`：缺陷处理规则。
- `HTML/CLAUDE.md`：HTML 规则。
- `C/CLAUDE.md`：C 语言规则。
- `Java/CLAUDE.md`：Java 规则。

## 用途

这些文件不再作为旧目录直接挂载，但内容已完整保留，供 `provider-guide` skill 在需要时引用。
